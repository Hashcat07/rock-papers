# Rock-Paper-Scissors Game

This is an interactive **Rock-Paper-Scissors** game built using HTML, CSS, and JavaScript. The game allows a human player to compete against the computer with a simple, responsive interface. Scores are tracked, and the game declares a winner when a player reaches 5 points.

---

## Project Overview

The game features:

- Three buttons for human player choices: Rock, Paper, and Scissors  
- Computer randomly selects its choice  
- Real-time score tracking for human and computer  
- Game ends when either the human or computer reaches 5 points, then the scores reset  
- Display of the winner once the game ends  

This project demonstrates fundamental JavaScript skills, including event handling, functions, conditionals, and DOM manipulation.

---

## Features

- Interactive buttons to select Rock, Paper, or Scissors
